# Ders Notu — Responsive Modül: Görseller Klasörü

| Alan | Açıklama |
|------|-----------|
| **Dersin konusu** | Responsive örnek HTML sayfalarına ait görsel varlıklar |
| **Kimler için?** | `Responsive/` altındaki ders dosyalarını kullanan öğrenciler |
| **Ön bilgi** | [Responsive modül ders notu](../EGITIM.md) |

---

## 1. Kazanımlar

- Duyarlı düzen örneklerinde görsellerin klasör yapısındaki yerini açıklamak.
- Farklı ekran genişliklerinde görsel yerleşiminin CSS ile nasıl yönetildiğine hazırlık.

---

## 2. Giriş

Responsive derslerde görseller bazen tam genişlik, bazen `max-width: 100%` ile akışkan davranış gösterir. Dosyaların bu alt klasörde toplanması, `Responsive` örneklerinin üst klasör görsellerinden ayrışmasına yardımcı olur.

---

## 3. Bu klasörün rolü

**Responsive** HTML dosyalarının ihtiyaç duyduğu **medya** dosyalarını saklar.

---

## 4. Kendini değerlendirme

Görsel çok büyük pikseldeyse mobilde hangi sorunlar yaşanabilir?

---

## 5. Uygulamalı çalışma

`1.media-query.html` veya `2.different-unit.html` ile birlikte bu klasördeki bir görselin yolunu takip edin.
